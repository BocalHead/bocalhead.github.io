Simon Says

15/09/2024

Un petit jeu simple

Le simon magique est un jeu populaire dont la plupart se souviennent de leur enfance. Le simon magique est un jeu de mémoire où l’on doit reproduire le même patterne que la machine. Une combinaison de lumières rouge, verte, jaune et bleu se déroule et nous devons la retenir pour la reproduire. C’est un jeu avec un algorithme relativement simple à reproduire, je me suis donc dit que ce serait un bon exercice de dév.

Comment ai-je fait ?

J’ai d’abord réalisé le modèle d’un jeu de simon magique. J’ai ensuite coloré les touches avec le shader graph intégré à Unity pour que les touches s’illuminent quand elles sont séléctionnées. Dans le code j’ai ensuite fait l’animation d’enfoncement de bouton avec du Tweening pour donner une animation de rebondissement lorsque le joueur appuie sur un bouton. J’ai ensuite procédé à la réalisation de l’algorithme qui donne un ordre et si l’ordre est respecté, le jeu répète l’ordre en rajoutant une touche en plus. J’ai fait un système de score assez anecdotique qui s’incrèmente lorsque l’on reproduit la bonne combinaison et un menu basique qui permet de choisir quand on relance le jeu ou pas après une partie.

Conclusion

C’est un jeu assez simple, mais je suis plutôt content du résultat. Le design de l’interface pourrait être grandement améliorée et je pourrais rajouter fonctions de qualité de vie comme la pause ou des paramètres. Mais le but était seulement de faire le jeu fonctionnel, donc cela peut attendre.
